## Projects

### 1. ATmega128을 사용한 안전한 전동 킥보드 시스템
**프로젝트 목표**  
전동 킥보드 이용 시 발생하는 안전 문제를 줄이기 위해, **헬멧 착용 / 음주 여부 / 양손 핸들링**의 3가지 조건을 모두 만족해야 운행할 수 있는 시스템을 구현했습니다.

**핵심 기능**
- 헬멧 착용 여부 확인
- 알코올 센서를 통한 음주 여부 측정
- 양손 핸들링 감지
- 헬멧부 ↔ 킥보드부 간 UART 통신
- 운행 중 안전 조건 실시간 재확인 및 위반 시 운행 종료
- 운행 시간, 요금, 자동 조명 등 운행 정보 관리

**사용 기술**  
`ATmega128` `MQ3 Sensor` `Touch Sensor` `Switch Sensor` `UART`

**내가 맡은 역할**
- 헬멧 착용 여부 확인 구현 **(10/10)**
- 음주 수치 측정 구현 **(10/10)**
- 양손 핸들링 확인 구현 **(5/10)**
- 헬멧부-킥보드부 UART 통신 구현 **(10/10)**

**문제 해결 경험**
- 머리카락 때문에 터치센서가 안정적으로 동작하지 않는 문제가 발생했습니다.  
  → 헬멧은 올바르게 착용했을 때 머리에 밀착되어야 한다고 판단해, **물리 스위치 센서 방식으로 변경하여 지속적인 착용 여부를 확인**했습니다.
- 헬멧부와 킥보드부 사이에서 여러 데이터를 각각 전송할 경우 지연이 발생했습니다.  
  → **알코올 수치와 헬멧 착용 데이터를 하나의 신호로 통합**하고 타이머를 활용해 전송 효율을 개선했습니다.

**한 줄 성과**  
최소 안전 조건을 만족하지 않으면 출발 자체가 불가능하고, 운행 중 위반 상황도 즉시 감지하는 **안전 중심 전동 킥보드 제어 시스템**을 구현했습니다.

**링크**
- 발표 자료 / 보고서: `여기에 링크 입력`
- 시연 영상: `여기에 링크 입력`

### 2. 객체 인식을 이용한 실시간 타겟 드론 팔로잉 시스템
**프로젝트 목표**  
타겟 드론을 실시간으로 인식하고 추적하여, 비행 중에도 안정적으로 따라가는 **드론 팔로잉 시스템**을 구현했습니다.

**핵심 기능**
- 타겟 드론 데이터셋 기반 객체 인식 모델 학습
- 전면 카메라 영상에서 타겟 드론 탐지
- Bounding Box 크기와 위치를 기반으로 거리 및 위치 추정
- ROS 토픽 발행을 통한 드론 위치/속도 제어
- 타겟 미인식 시 호버링 유지

**사용 기술**  
`YOLOv5` `ROS` `PX4` `Gazebo` `Ubuntu` `Jetson Nano` `Intel RealSense`

**내가 맡은 역할**
- 모델 학습 **(8/10)**
- 위치 제어 알고리즘 구현 **(8/10)**
- 전체 파이프라인 구성 **(10/10)**
- 최적화 **(10/10)**

**문제 해결 경험**
- Jetson Nano에서 이미지 분석부터 위치 제어까지 이어지는 과정에서 버퍼링이 발생하고 인식률이 낮아졌습니다.  
  → **센서 스트리밍 구성과 카메라 프레임 수를 조정**해 처리 부담을 줄이고 반응성을 개선했습니다.
- 드론 기체의 진동과 불안정한 비행 때문에 통신까지 불안정해졌습니다.  
  → **하드웨어 결합 상태를 보강**해 비행 안정성과 통신 안정성을 함께 개선했습니다.
- 제어 방식을 속도 기반으로 할지 위치 기반으로 할지 팀 내 의견 차이가 있었습니다.  
  → 두 방식을 직접 테스트한 뒤, 실제 비행에서 더 안정적인 **속도 제어 방식**을 선택했습니다.

**한 줄 성과**  
시뮬레이션 환경에서 **80% 이상의 인식 정확도**를 확인했고, 실제 비행에서도 타겟 드론 추적 및 미인식 시 호버링 동작을 구현했습니다.

- 논문 / 보고서: `여기에 링크 입력`
- 발표 자료: `여기에 링크 입력`

### 3. Facial Emotion Detect 노래 추천 웹사이트
**프로젝트 목표**  
사용자의 얼굴 표정을 인식해 감정을 예측하고, 그 결과에 맞는 음악을 추천하는 **감정 기반 음악 추천 웹서비스**를 구현했습니다.

**핵심 기능**
- 얼굴 표정 인식 기반 감정 예측
- 사용자 얼굴 인식 및 개인별 기록 관리
- 감정 결과 기반 Spotify 음악 추천 및 자동 재생
- 웹페이지에서 현재 감정과 과거 기록 확인
- 사용자 이용 내역 DB 저장

**사용 기술**  
`ResNet18` `K-NN` `Flask` `SQLite3` `Spotify API` `face_recognition`

**내가 맡은 역할**
- 감정 예측 모델 학습 **(10/10)**
- 사용자 인식 모델 및 데이터베이스 구축 **(10/10)**
- 웹페이지 구현 **(5/10)**

**문제 해결 경험**
- 초기 표정 인식 모델의 정확도가 낮았습니다.  
  → 기존 데이터셋이 서양인 및 흑백 이미지 중심이라는 점을 분석한 뒤, **동양인 비중이 높은 데이터셋을 추가**하고, 클래스 불균형을 반영해 **가중치 조정과 데이터 증강**을 적용했습니다.
- 사용자 식별 과정에서 딥러닝 기반 재학습 방식은 시간과 데이터 측면에서 비효율적이었습니다.  
  → **K-NN 알고리즘을 적용해 빠르게 사용자 데이터를 구축하고 인식할 수 있도록 설계**했습니다.

**한 줄 성과**  
감정 예측, 사용자 식별, 기록 저장, 음악 추천까지 연결되는 **AI 기반 감정 맞춤형 음악 추천 웹서비스**를 구현했습니다.
