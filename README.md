### 🛴 ATmega128을 사용한 안전한 전동 킥보드 시스템

**Overview**  
전동 킥보드 이용 시 발생하는 안전 문제를 줄이기 위해, 헬멧 착용 여부·음주 여부·양손 핸들링을 모두 확인한 뒤에만 운행이 가능하도록 설계한 안전 중심 시스템입니다.

**Key Features**
- 헬멧 착용 여부 확인
- MQ3 센서를 활용한 음주 수치 측정
- 손잡이 양쪽 터치 센서를 통한 양손 핸들링 감지
- 헬멧부 ↔ 킥보드부 간 UART 통신
- 운행 중 안전 조건 실시간 재확인 및 위반 시 운행 종료
- 운행 시간, 요금, 자동 조명 등 운행 정보 관리

**Tech**  
<img src="https://img.shields.io/badge/ATmega128-1F2937?style=flat-square" alt="ATmega128" />
<img src="https://img.shields.io/badge/MQ3%20Sensor-1F2937?style=flat-square" alt="MQ3 Sensor" />

<img src="https://img.shields.io/badge/UART-1F2937?style=flat-square" alt="UART" />

**My Contributions**
- 헬멧 착용 여부 확인 구현 `(10/10)`
- 음주 수치 측정 구현 `(10/10)`
- 양손 핸들링 확인 구현 `(5/10)`
- 헬멧부-킥보드부 UART 통신 구현 `(10/10)`

**Troubleshooting**

### 🚁 객체 인식을 이용한 실시간 타겟 드론 팔로잉 시스템

**Overview**  
타겟 드론을 실시간으로 인식하고, bounding box 정보와 ROS 토픽 발행을 활용해 추적 드론의 위치와 속도를 제어하는 드론 팔로잉 시스템입니다.

**Key Features**
- 타겟 드론 데이터셋 기반 객체 인식 모델 학습
- 전면 카메라 영상에서 타겟 드론 탐지
- Bounding box 크기와 좌표를 활용한 위치 조정
- ROS 토픽 발행을 통한 드론 제어
- 타겟 미인식 시 호버링 유지
